# Docker

## Objetivo

Preparar servicios locales para desarrollo.

## Servicios previstos

- Web Astro.
- Admin Angular.
- API.
- Worker.
- Redis.
- Supabase local opcional.

## Desarrollo

El lanzamiento principal del proyecto es:

```bash
npm run dev
```

Docker queda reservado para Redis, entorno local completo o staging.
