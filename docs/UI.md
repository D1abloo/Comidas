# UI — Bocado Delivery

## Estilo

- Minimalista editorial.
- Fondo blanco cálido.
- Navegación negra.
- Acento amarillo lima.
- Tipografía sans serif moderna.
- Cards redondeadas.
- Sombras suaves.
- Mucho espacio en blanco.

## Home pública

Debe incluir:

- Header con logo.
- Hero con CTA.
- Filtros por cocina, tiempo, precio y categoría.
- Cards de platos.
- Carrito.
- Checkout.
- Sección de confianza.
- Footer con logo, enlaces y legales.

## Cards de platos

Cada card debe mostrar:

- Imagen realista.
- Restaurante.
- Nombre del plato.
- Precio.
- Tiempo estimado.
- Valoración.
- Badge de disponibilidad.
- Botón Añadir.
- Favorito.

## Admin

Debe incluir:

- Dashboard.
- Métricas.
- Gráfico de ventas.
- Tabla de pedidos.
- Gestión de platos.
- Gestión de pagos.
- Gestión de facturas.
- Configuración de empresa.
- Configuración de Bizum.
- Notificaciones por email y WhatsApp.

## Estados visuales

- Pedido pendiente.
- En preparación.
- En reparto.
- Entregado.
- Pago pendiente.
- Pago confirmado.
- Factura enviada.
- Bizum pendiente de confirmar.
